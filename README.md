1. Data Crawling menggunakan Tweet-Harvest
2. Data Preparation
3. Data Preprocessing
4. Word Embedding IndoBERTweet
5. Modeling menggunakan Decision Tree
6. Evaluasi Confusion Matrix
7. Visualisasi Sentimen
